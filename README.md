## 📱 نبذة عن التطبيق

**Nour** (نور) هو تطبيق ويب ذكي يساعدك في تخطيط رحلاتك بسهولة واحترافية. مع واجهة عربية وإنجليزية، وميزات متقدمة للتخطيط والحجز.

### ✨ الميزات الرئيسية

- 🤖 **مساعد ذكي AI** - محادثة طبيعية لتخطيط الرحلة
- 🗺️ **خرائط تفاعلية** - عرض الوجهات على الخريطة
- 💰 **إدارة الميزانية** - تتبع تفصيلي للنفقات
- 📅 **جدول يومي ذكي** - برنامج مقترح للأنشطة
- 🎲 **مولد الوجهات العشوائي** - اكتشف وجهات جديدة
- 🌍 **دعم لغات متعددة** - عربي/إنجليزي
- 📱 **تطبيق ويب تقدمي (PWA)** - يعمل أوفلاين
- 💳 **نظام الدفع** - تكامل Stripe
- 🔐 **حسابات مستخدمين** - تسجيل وتحقق آمن
- 📊 **لوحة تحكم** - إحصائيات وتقارير
